A plugin to enable easy integration of Digital Goods Store buy now and add to cart buttons into a WordPress post or page with [dgs] shortcodes.

== Description ==

Sell downloads on your WordPress powered site with ease! Install this plugin to enable easy integration of the Digital Goods Store service http://www.digitalgoodsstore.com buy now and add to cart buttons into a WordPress post or page with [dgs] shortcodes.

For a graphical overview of the installation and usage visit:- http://www.digitalgoodsstore.com/sell-downloads-with-wordpress

== Installation ==

1. Upload `Digital-Goods-Store-Button-Generator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste the [dgs] shortcode generated by the Digital Goods Store button wizard into WordPress posts or pages where you want your buy now or add to cart buttons to appear.
